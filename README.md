Media_Mint_Assignment

This project contains a frontend (React Vite) and a backend (Django).

Clone the project and follow the steps below.

Project Contains

Frontend: React (Vite)

Backend: Django

STEPS TO RUN
Backend

A. cd backend
B. pip install -r requirements.txt
C. Activate virtual environment (optional if you created one)
D. python manage.py runserver 8000

Frontend
cd frontend
npm install
npm run dev


(Click the link shown in the terminal)

------------------------------------ Implementation ------------------------------------

First, the user must register in the Register page.
(Register at least 3 users for testing)

Login using one of the user credentials.

Now the user can create a Project.

By default, if a user creates the project, they become the Maintainer of that project.

You can create as many projects as you want.

Bug: You need to reload the page to see the newly created project.

Click on any project.

You can create Issues inside the project.

Once you create an Issue, you can assign it to any user (that user is added as a member of the project).
Bug: You need to reload the page to see the newly created Isuue.

You can log in using the user you assigned the project to.

The project will appear under that user, but as a Member.

Maintainer can edit the Issue.
Member can only view the Issue and comment on it.
